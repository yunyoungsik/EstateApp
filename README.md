## 설치

```
api
npm i express
npm i nodemon
npm i bcrypt
npm i prisma
npx prisma init --datasource-provider mongodb
npm i @prisma/client
npm i cookie-parser
npm i jsonwebtoken
npm i cors

prisma
npx prisma db push

client
npm i axios
npm i react-quill
npm i dompurify
npm i timeago.js
npm i socket.io-client
npm i zustand

socket
npm i nodemon
npm i socket.io
```
